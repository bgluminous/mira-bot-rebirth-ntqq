# Mira Bot Rebirth NTQQ

## 简介

这是一个基于 Java 开发使用 onebot-11 协议对接 LLOneBot 的 SDK

目的是为了加快 Java 开发者的开发速度

工具类已经集成了 WebSocket 的正向反向连接控制以及事件的解析和发送工具类

开发者只需要专注于逻辑编写即可

(理论上只要是基于onebot-11协议的实现都可以对接但是不提供保证)
